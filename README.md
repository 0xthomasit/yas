# YAS: Yet Another Shop

YAS is a pet project aim to practice building a typical microservice application in Java

## Intended technologies and frameworks

- Java 17
- Spring boot 2.6
- Next.js
- Keycloak
- Kafka
- K8s
- Github Action
- SonarCloud

## Components roadmap
- [ ] Identity service
- [ ] Product service
- [ ] Pricing service
- [ ] Cart service
- [ ] Order service
- [ ] Review service
